  1. 获得所有接口

    说明 用于获得所有接口数据
    路由
    方法 URI
    GET /api/InterfaceInfo
    参数 无
    返回值
    名 值类型 解释
    status string 状态值，参考:附录1
    list array[json] 接口列表1

    1 接口列表json
    名 类型 必需 解释
    InterfaceName string 1 项目名
    FunctionName string 1 方法名
    ReturnType string
    ReturnName string
    ReturnTemplateId string
    ParaViewModels array[json] 接口字段列表1.1

    1.1 接口字段列表json
    名 类型 必需 解释
    ParaName string 1 字段名
    ParaType string 1 字段类型
    ParaIndex string
    示例
    [
    "InterfaceName":"Intel",
    "FunctionName":"func",
    "ReturnType":"0001",
    "ReturnName":"return",
    "ReturnTemplateId":"dbkasbd",
    "ParaViewModels":[
    ["ParaName":"p1", "ParaType":"0002", "ParaIndex":"1"],
    ["ParaName":"p1", "ParaType":"0002", "ParaIndex":"1"],
    ["ParaName":"p1", "ParaType":"0002", "ParaIndex":"1"]
    ]
    ]
